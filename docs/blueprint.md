# **App Name**: Community Compass

## Core Features:

- User Authentication: Secure login for field workers and administrators using Firebase Authentication.
- Household Registration: Capture household details including GPS coordinates using Capacitor's Geolocation API.
- Child Data Entry: Record child-specific information and progress updates.
- Photo Capture: Capture family and house photos using the native camera API (Capacitor).
- Progress Monitoring Tool: LLM analyzes child progress updates to identify trends and challenges, offering insights for intervention programs and summarizing it's findings.
- Automated Follow-up Scheduling: Schedule recurring visits and generate alerts for overdue follow-ups.
- Data Visualization and Reporting: Generate charts and reports from aggregated data for analysis and export.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke growth and stability.
- Background color: Soft sage (#E8F5E9), a desaturated variant of green to complement primary.
- Accent color: Warm yellow (#FDD835), creating contrast and signaling important actions.
- Headline font: 'Poppins' (sans-serif) for a contemporary, geometric feel.
- Body font: 'PT Sans' (sans-serif) for readability and warmth in longer texts. To be paired with Poppins.
- Use clear, descriptive icons for navigation and data representation.
- Maintain a clean, intuitive layout optimized for mobile and web interfaces.
- Subtle transitions and feedback animations to enhance user experience.
